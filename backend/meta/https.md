# Check that VM accepts external http and HTTPS

Google Cloud Console -> open VM page -> `Edit` -> `Firewalls` -> Click `Allow HTTP traffic` and `Allow HTTPS traffic`.

# Get static external IP

Google Cloud Console -> search `ip addresses` -> `External IP Addresses` -> find the one the VM uses -> promote to static -> copy the IP

# Download lego on the VM

Download lego release from https://github.com/go-acme/lego, and put `lego` in `/usr/local/bin`

# Change nginx temporarily to serve the future certs

Set `/etc/nginx/sites-available/default` to:

```
server {
    listen 80 default_server;
    listen [::]:80 default_server;
    server_name _;

    location /.well-known/acme-challenge/ {
        root /var/www/html;
    }

    location / {
        return 301 https://$host$request_uri;
    }
}
```

Then run `sudo nginx -t && sudo systemctl reload nginx`.

# Generate certs

Run while CWD is `/home/me`:

```shell
sudo lego run --email <your email> \
    --server https://acme-v02.api.letsencrypt.org/directory \
    --profile shortlived \
    --http --http.webroot /var/www/html \
    --domains <VM external IP>
```

# Backup the directory

VM:
```shell
sudo su
cd /home/me/.lego
tar -cvf ../archive.tar accounts
```

PC:
```shell
gcloud compute scp me@instance-20260502-221724:/home/me/archive.tar --zone=us-central1-f .
```

VM:
```shell
rm ../archive.tar
```

# Add auto-renewal

Copy `./myLegoRenew.service` and `./myLegoRenew.timer` to `/etc/systemd/system/` (change the service to have your IP).

Then `sudo systemctl daemon-reload && sudo systemctl enable --now myLegoRenew.timer`.

# Update nginx to new configuration

Set `/etc/nginx/sites-available/default` to:

```
server {
    listen 80 default_server;
    listen [::]:80 default_server;
    server_name _;

    location /.well-known/acme-challenge/ {
        root /var/www/html;
    }

    location / {
        return 301 https://$host$request_uri;
    }
}

server {
    listen 443 ssl http2;
    listen [::]:443 ssl http2;
    server_name <VM external IP>;

    ssl_certificate     /home/me/.lego/certificates/<VM external IP>.crt;
    ssl_certificate_key /home/me/.lego/certificates/<VM external IP>.key;


    ssl_protocols TLSv1.2 TLSv1.3;
    ssl_ciphers HIGH:!aNULL:!MD5;
    ssl_prefer_server_ciphers on;
    ssl_session_cache shared:SSL:10m;
    ssl_session_timeout 1d;

    location / {
        proxy_pass http://127.0.0.1:3000;
        proxy_http_version 1.1;

        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";

        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        proxy_set_header X-Forwarded-Host $host;

        proxy_read_timeout 60s;
        proxy_send_timeout 60s;
    }
}
```

Then run `sudo nginx -t && sudo systemctl reload nginx`.
